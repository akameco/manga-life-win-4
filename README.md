# manga-life-win-4

> fetch links from mangalifewin

## Installation

```
$ npm install --save manga-life-win-4
```

## Usage

```js
'use strict';
const mlw4 = require('manga-life-win-4');

module.exports = () => {
	return mlw4('http://mangalifewin.takeshobo.co.jp/rensai/popute2/');
};
```
